# AI-Assisted Skin Disease Detection System

## Overview

The AI-Assisted Skin Disease Detection System is a web-based application that uses deep learning and symptom analysis to assist users in identifying common skin diseases. The system allows users to upload an image of a skin condition or enter symptoms to receive a preliminary prediction along with disease information, precautionary measures, and treatment suggestions.

**Note:** This application is intended for educational and research purposes only. It is not a substitute for professional medical diagnosis or treatment.

---

## Features

- Image-based skin disease detection using a trained CNN model
- Symptom-based disease prediction using rule-based logic
- AI chatbot for answering skin health-related queries
- Multilingual interface (English, Hindi, Kannada)
- Downloadable PDF medical report
- User-friendly and responsive web interface

## Workflow

1. User uploads a skin image or selects symptoms.
2. The frontend sends the request to the FastAPI backend.
3. The backend preprocesses the image.
4. The trained CNN model predicts the disease.
5. Symptom-based logic provides an additional prediction when applicable.
6. Results, confidence score, precautions, and treatment suggestions are displayed.
7. Users can interact with the chatbot or download a PDF report.

## Installation

### Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/AI-Assisted-Skin-Disease-Detection-System.git
cd AI-Assisted-Skin-Disease-Detection-System
```

### Backend Setup

```bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend Setup

```bash
cd frontend
npm install
npm start
```
